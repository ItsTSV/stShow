# IT: Streamlit Showcase for ML/AI Web Applications

This project serves as a **technical demonstration** for the Internet Technologies course. The primary goal is to showcase a wide range of Streamlit components and capabilities within a single .py file.
In real world, it would be better to split the code into multiple files, packages, functions etc. for better maintainability and readability. 

### Features
- Multipage navigation
- Interactive EDA to show usage of charts, dataframes and dynamic filters
- Map visualisation with custom markers
- Live data fetching with https requests and APIs
- AI model inference
- Caching

### Commands
Get all dependencies:
```
pip install -r requirements.txt
```

Run the app:
```
streamlit run src/app.py
```

Linting:
```
ruff format
```
